Developed a machine learning model for music genre classification using the K-Nearest Neighbors (KNN) algorithm. The system analyzes low-level audio features such as tempo, spectral centroid, zero-crossing rate, and Mel-Frequency Cepstral Coefficients (MFCCs), along with high-level rhythm and harmonic patterns, to accurately classify songs into genres like rock, jazz, classical, pop, hip-hop, and electronic. The workflow begins with audio preprocessing, where tracks are segmented and normalized to reduce noise and standardize volume levels. Using the Librosa library, a comprehensive set of features is extracted from each audio sample, which is then scaled and fed into the KNN model. Different values of 'k' and distance metrics (Euclidean, Manhattan) were tested to optimize accuracy. Extensive experiments were conducted on benchmark datasets such as the GTZAN music genre dataset. The model achieved competitive accuracy compared to other baseline classifiers, demonstrating the effectiveness of simple distance-based approaches in music information retrieval. Confusion matrices and cross-validation were employed to analyze misclassifications and improve feature selection.

Leveraging Librosa for audio feature extraction and scikit-learn for training, the model achieved strong classification performance. This project demonstrates how machine learning can be applied to enhance music recommendation systems and audio-based applications.
